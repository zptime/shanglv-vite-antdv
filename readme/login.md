用户登录

拦截路由变化
↓

- 判断 vuex 中是否有用户信息
  ↓
  - 有用户信息
    ↓
    - 验证 token
      ↓ - token 验证通过
      ↓ - 跳转路由
      ↓ - token 验证不通过
      ↓ - 跳转登录页面
      ↓
  - 没有用户信息
    ↓
    - 判断 localStorage 中是否有用户信息
      ↓
      - 有用户信息
        ↓
        - 验证 token
          ↓ - token 验证通过
          ↓ - 获取 localStorage 用户信息
          ↓ - 设置 vuex 中的用户信息
          ↓ - 重新设置用户路由
          ↓ - token 验证不通过
          ↓ - 跳转登录页面
          ↓
      - 没有用户信息
        ↓
        - 跳转登录
